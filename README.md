# Tip Calculator

Desafio proposto no curso da b7web, com intuito de colocar em prática os estudos no reactJs, biblioteca mais popular do JavaScript e é usada para construir uma interface de usuário. Ela oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.
